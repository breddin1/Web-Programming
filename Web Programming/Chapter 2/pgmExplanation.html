Circle Radius

Below is the following program on how to find
the radius of a circle using code!

int main()
{
double pi = 3.14, circumference, radius, area;

cout << "Please enter the radius of the circle in inches: ";
cin >> radius;

circumference = 2 * pi * radius ;

area = pi * radius*radius;

cout << "\n\n";

cout << "Radius: " << radius << endl;

cout << "Circumference: " << circumference << endl;

cout << "Area: " << area << endl;

return 0;
}

The variable radius has been given the bold font to help point out its input
and its output at the end of the program.

You can see at the beginning of the code we are declaring the varible radius as double or float.
This is to allow the numbers to deal with decimals, example being 1.02. This is crucial to the program.
We will not consider circle radius as C.R.
